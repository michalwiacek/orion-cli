Orion CLI

OpenAPI-first CLI for humans and AI agents

Discover endpoints, inspect request and response details, execute operations, and generate ready-to-run curl commands from operation IDs.

View on GitHub Read Docs

Install

zig build install --prefix ~/.local

Build & Test

zig build
zig test src/openapi_loader_integration_test.zig

Quickstart

orion list
orion describe get:/health
orion call get:/health
orion curl get:/health -k --pretty

Config

Layered config: project overrides global.

  • ~/.config/orion/config.json
  • .orion/config.json
